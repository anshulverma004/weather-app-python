# Weather App Using API (Console Based)

A simple console-based Weather Application built using Python that displays real-time weather information for any city entered by the user. This project was developed as part of **Taskâ€“03** during my internship at **IncodeVision**.

## Features
- Enter city name as user input
- Fetches real-time weather data using a weather API
- Displays temperature, humidity, weather condition, and wind speed
- Handles invalid city names and API errors
- Clean and simple console output

## Technologies Used
- Python
- Requests module
- REST API
- JSON data handling
